# WooCommerce Custom Templates

Custom PHP templates and components built for a live greenhouse e-commerce client on WooCommerce. Includes full product page templates, cart hooks, and reusable sections.

**Live demos:** [ryliewalsh.github.io/woocommerce](https://ryliewalsh.github.io/woocommerce)

## Tech
PHP · WooCommerce API · WordPress Hooks · Vanilla JS · CSS Grid · PostgreSQL

## Structure
- `src/` — original PHP source files deployed to production
- `demos/` — standalone HTML demos (no WordPress install required)
